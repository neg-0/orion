# Orion Documentation

This directory contains comprehensive documentation for the Orion project, structured to be both human-readable and AI-agent-friendly.

## Directory Structure

```
docs/
├── architecture/           # System architecture and design decisions
├── agents/                # AI agent specifications and behaviors
├── mental-model/          # Project understanding and context
├── workflows/             # Process and automation workflows
├── specifications/        # Detailed technical specifications
└── examples/             # Usage examples and patterns
```

## Documentation Guidelines

- All documentation should be in markdown format
- Include metadata headers for AI agent context
- Use consistent terminology across documents
- Link related concepts and documents
- Include rationale for important decisions
